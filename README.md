# 國泰國小 蘇國瑞主任六年級社會領域公開課研究成果

本專案為新北市新莊區國泰國小學務處主任 **蘇國瑞主任** 六年級社會領域公開課的課例研究與簡報成果。我們基於**學習共同體 (Study of Learning Community, SLC)** 的「描述-詮釋-反思」框架，對公開課影片進行了深度語音轉譯、關鍵畫面擷取、AI 插畫重繪，並自動排版生成了寬螢幕簡報。

## 🔍 課例分析亮點摘要

本堂公開課主題聚焦於 **「國泰校史與新莊地方產業變遷探究」**。學生分成四組，透過小白板外顯化思考，整理並發表姚素蓮前校長與葉錫東校友的專訪印象。

1.  **描述 (Description)**：
    *   學生以順時針輪流發表，在組內凝聚共識後，將小白板張貼於黑板上發表最獨特的記憶。
2.  **詮釋 (Interpretation)**：
    *   **校史危機**：學生驚訝於學校曾因人數銳減面臨廢校危机，以及校舍（和平、愛國、中孝、信義樓）與校地從西盛遷移至現址的演變。
    *   **產業與生命史**：學生將葉錫東校友童年機器夾傷手的故事，與溫仔圳「洪水平原管制區」禁建、鐵皮屋工廠林立及製冰廠粉塵等地方工業史相連結。
3.  **反思 (Reflection)**：
    *   **耳朵打開的聆聽關係**：蘇主任引導學生建立聆聽關係，互相向他人學習。
    *   **織網學習**：小組探究讓個人故事與宏觀社會結構形成有機連結，是一堂精彩的社會領域學習共同體實踐課。
